About Me

I'm a software engineer, currently writing code, mostly Java, for a financial company. I completed my PhD in Computer Science in 2014 (dissertation title: "Development and User Testing of New User Interfaces for Mathematics and Programming Tools"--more on this work here). I am particularly passionate about writing reliable, bug-free programs that clearly do what they clearly say they do, and finding ways to make using computers more efficient for more tasks.
